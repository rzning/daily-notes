# 画人物

人物的三个特征

- 人物的性格
  - 五官、神态、动作
- 职业
  - 衣着、场景
- 细节

示例：画萌妹

- 性格：大眼睛、圆脸、可爱、活泼、粘人、肉嘟嘟
- 职业：学生、JK
- 细节：酒窝、眨眼、长发、背包、双马尾、双眼皮、卷发、长睫毛
