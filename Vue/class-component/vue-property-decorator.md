Vue Property Decorator

## Install

- peerDependencies
  - `vue`
  - `vue-class-component`

```sh
yarn add vue-property-decorator
```

## Usage

- `@Prop`
- `@PropSync`
- `@Model`
- `@Watch`
- `@Provide`
- `@Inject`
- `@ProvideReactive`
- `@InjectReactive`
- `@Emit`
- `@Ref`


<section>

<details>
<summary>@prop</summary>

```js
var aaa = "aaaa"
```

</details>

</section>
