# Vetur

Vue tooling for VS Code, powered by [vue-language-server]

[vue-language-server]: <https://github.com/vuejs/vetur/tree/master/server>

## Features

- 语法高亮
- 片段
- Emmet
- 整理/错误检查
- 格式化
- 智能感知
- 调试
- 框架支持
- 插值支持
- VTI / CLI
