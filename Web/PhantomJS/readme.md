# PhantomJS

- <http://phantomjs.org/>
- <https://github.com/ariya/phantomjs>

PhantomJS 是一个支持 JavaScript 脚本的 headless 网页浏览器。

使用 QtWebKit 作为后端，为各种 web 标准提供快速和本机支持：

> DOM handling, CSS selector, JSON, Canvas, and SVG.
