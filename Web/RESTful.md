# RESTful

- Representational State Transfer
- 表述性状态传递

REST 是 Roy Fielding 博士在 2000 年他的博士论文中提出来的一种软件架构风格。

REST 是一种网络应用程序的设计风格和开发方式，基于 HTTP，可以使用 XML 格式定义或 JSON 格式定义。
