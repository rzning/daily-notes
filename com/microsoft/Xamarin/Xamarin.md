# Xamarin

- <http://xamarin.com>
- <https://dotnet.microsoft.com/apps/xamarin>
- <https://forums.xamarin.com/>

Xamarin 是一个开源平台，通过 .NET 构建适用于 iOS、Android 和 Windows 的新式应用程序。

Xamarin.Forms 允许你使用采用 C# 或 XAML 编写的共享 UI 代码生成本机应用。
