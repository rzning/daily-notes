Emscripten: An LLVM-to-Web Compiler

Emscripten 是用于编译为使用 LLVM 构建的 asm.js 和 WebAssembly 的工具链，
可让您以几乎本机的速度在 Web 上运行 C 和 C++ 而无需插件。
